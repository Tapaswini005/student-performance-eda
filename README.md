# 📊 Exploratory Data Analysis (EDA) – Student Performance Dataset

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a student performance dataset to understand the key factors affecting academic results.

The analysis includes data cleaning, visualization, and interpretation of patterns using Python libraries.

---

## 📂 Dataset Description

The dataset contains information about students, including:

* Study Hours Per Day
* Attendance Percentage
* Subject Scores (Math, Science, English)
* Previous Year Score
* Final Percentage
* Performance Level
* Pass/Fail Status

📊 Total Records: **5000**
🧾 Data Type: Structured (Numerical + Categorical)

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

---

## 🔍 Steps Performed

1. Data Loading
2. Data Cleaning

   * Checked missing values
   * Removed duplicates
3. Data Exploration

   * Statistical summary
   * Feature understanding
4. Data Visualization:

   * Histogram
   * Count Plot
   * Scatter Plot
   * Line Chart
   * Pie Chart
   * Box Plot
   * Heatmap

---

## 📈 Key Visualizations

* Distribution of Final Percentage
* Gender vs Pass/Fail comparison
* Study Hours vs Final Score relationship
* Average Score trend with study hours
* Performance level comparison using boxplots
* Correlation analysis using heatmap

---

## 💡 Key Insights

* 📌 Study hours have a strong positive impact on final scores
* 📌 Attendance significantly affects performance
* 📌 Previous academic performance influences final results
* 📌 High-performance students consistently score higher
* 📌 Most students fall in the pass category

---

## ✅ Conclusion

The analysis reveals that **consistent study habits, good attendance, and past academic performance** are the most important factors influencing student success.
